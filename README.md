## 📚 What's Inside?

### Python 🐍
- **HI.py** - A strange greeting using only boolean True
- **longest_hello_world.py** - The most unnecessarily complex "Hello World" ever
- **quantum_object.py** - Objects in superposition (until observed!)
- **quine.py** - A program that outputs its own source code
- **remember_state.py** - Functions that remember their state
- **shapeshifter.py** - Objects that change types dynamically

### C 🔧
- **fast_inverse_sqrt.c** - The famous Quake III inverse square root "hack"
- **negative_indexing.c** - Accessing memory in weird ways

### C++ 🎭
- **cursed_cpp_tokens.cpp** - Digraphs and alternative tokens

### JavaScript 🚀
- **js_coercion_madness.js** - Type coercion nightmares and wonders

## 🚀 Getting Started

Each file can be run independently:

```bash
# Python
python3 python/HI.py
python3 python/quine.py

# C
gcc c/fast_inverse_sqrt.c -o fast_sqrt && ./fast_sqrt
gcc c/negative_indexing.c -o neg_index && ./neg_index

# C++
g++ cpp/cursed_cpp_tokens.cpp -o cursed && ./cursed

# JavaScript
node javascript/js_coercion_madness.js
```
